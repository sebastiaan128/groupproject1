# GroupProject1 – Studentenplatform

## Project Omschrijving

GroupProject1 is een webapplicatie gebouwd met het Symfony PHP-framework, bedoeld als sociaal leerplatform voor studenten. Het platform stelt studenten in staat om:

- **Profielen aan te maken** met persoonlijke gegevens zoals naam, e-mailadres, studierichting, studiejaar, bio en kennistags.
- **Vragen te stellen en te beantwoorden** via een Questions & Answers-systeem, waarbij vragen gefilterd kunnen worden op status (open, beantwoord, eigen vragen).
- **Een netwerk op te bouwen** door verbinding te maken met andere studenten.
- **Groepschats te gebruiken** om met medestudenten te communiceren.

De applicatie maakt gebruik van **Firebase** voor realtime functionaliteit, **Doctrine ORM** voor databasebeheer en **Tailwind CSS** voor de stijl.
